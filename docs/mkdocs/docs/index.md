This site provides details about ``robotframework-testdoc`` - the new tool for generating test documentations for your Robot Framework project.
It supports modern tools like ``jinja2`` & ``mkdocs`` to design good readable & good looking webpages!

## Installation

Use the following command to install the tool:
```shell
pip install robotframework-testdoc
```
